## 1. Exam Information

* Duration
  * 2 hours, 16 questions
* Allowed Reference Sites
  * https://kubernetes.io/docs/home/
  * https://github.com/kubernetes/
  * https://kubernetes.io/blog/
  * Other sites are not allowed
* Exam Environment
  * Kubernetes v1.21

## 2. Exam Preparation

* Required Materials
  * Chrome Browser
  * Webcam
  * Microphone
  * Passport
* Exam Environment Check
  * https://www.examslocal.com/ScheduleExam/Home/CompatibilityCheck

## 3. Pre-Exam Checklist

* Verify kubectl bash autocompletion
  * https://kubernetes.io/docs/reference/kubectl/cheatsheet/
* Verify tmux functionality
  * https://linuxize.com/post/getting-started-with-tmux/

## 4. Commands to Know During the Exam

* Windows Copy
  * ctrl + insert
* Windows Paste
  * shift + insert

## 5. Exam Review

* Practicing 2-3 times on Kode Cloud with concept understanding should be sufficient to pass
  * https://kodekloud.com/courses/certified-kubernetes-application-developer-ckad/
