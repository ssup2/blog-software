## 1. Kubernetes Ephemeral Storage

Ephemeral Storage는 일시적인 데이터를 저장하는 데 사용되는 스토리지 타입이다. 이 스토리지는 데이터가 임시로 저장되고, 필요한 경우 삭제될 수 있다.
