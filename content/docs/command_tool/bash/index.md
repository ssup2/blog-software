Bash Shell 사용법을 정리한다.

## 1. Bash

### 1.1. Cursor 이동 

* ctrl + f : 한 글자 앞으로 이동
* ctrl + b : 한 글자 뒤로 이동
* ctrl + a : Line의 가장 앞으로 이동
* ctrl + e : Line의 가장 뒤로 이동
* alt + f : 한 단어 앞으로 이동
* alt + b : 한 단어 뒤로 이동

### 1.2. 삭제

* ctrl + w : 한 단어 삭제

### 1.3. E.T.C

* ctrl + - : Undo
